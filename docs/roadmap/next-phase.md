# Next Phase Roadmap

## W9：生产化状态与配置中心
- 把共享记忆、实验配置、发布状态迁移到持久化后端。
- 建立统一配置中心，支持实验开关、阈值、白名单动态更新。
- 增加跨实例恢复与状态追踪。

## W10：真实基础设施接入
- 接入真实消息总线、任务队列与分布式调度。
- 把多 Agent 协作从单进程骨架升级为可横向扩展架构。
- 接入真实 metrics、logs、traces 后端。

## W11：生产级评测与回放
- 建立脱敏流量回放体系。
- 把 A/B、灰度、故障注入纳入自动化回归。
- 增加线上收益、成本、满意度联合评估。

## W12：治理闭环增强
- 建立发布控制台、值班面板、事故时间线面板。
- 增加自动止损策略和推荐处置建议。
- 形成周度运营报表与错误预算管理机制。

## 里程碑
| 阶段 | 目标 | 完成标志 |
|------|------|----------|
| M1 | 状态生产化 | 支持持久化与跨实例恢复 |
| M2 | 基础设施接入 | 接入消息队列与真实观测后端 |
| M3 | 自动化治理 | 支持回放、演练、自动止损 |
| M4 | 运营闭环 | 支持周报、预算、发布控制台 |
