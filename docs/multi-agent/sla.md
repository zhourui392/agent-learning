# Multi-Agent SLA

## 响应目标
- `planner`：1 秒内完成任务拆解。
- `executor`：5 秒内返回单步执行结果。
- `auditor`：3 秒内完成结果复核。
- `coordinator`：1 秒内完成汇总与状态推进。
- `human`：15 分钟内响应高优先级介入请求。

## 错误处理责任
- `planner`：任务定义不清、缺关键上下文时负责回退重规划。
- `executor`：工具失败、数据不足时负责回传结构化错误码。
- `auditor`：发现冲突、证据不足时负责触发仲裁或人工兜底。
- `coordinator`：任务超时、回传缺失时负责重试和回收。

## 升级条件
- 单任务重试超过 1 次。
- 冲突仲裁进入 `needs_human`。
- 共享记忆连续出现版本冲突。
